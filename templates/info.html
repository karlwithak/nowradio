About {{ config['APP_NAME'] }}

{{ config['APP_NAME'] }} is a place where you can find internet radio now.
If you have any questions or comments please send them to info@nowrad.io

Keyboard controls

  • Space bar: play/pause
  • left arrow key: previous station in current genre
  • right arrow key: next station in current genre
  • double-tap left arrow key: previous genre
  • double-tap right arrow key: next genre
  • up arrow key: increase volume
  • down arrow key: decrease volume
  • s key: add current station to favourites
  • 1-5 number keys: start playing corresponding favourite
